# 项目文档索引

本目录集中存放课程提交、设计说明、接口说明、测试记录和验收截图材料。

## 核心文档

| 文档 | 说明 |
| --- | --- |
| [课程报告.md](./课程报告.md) | 按课程任务书撰写的 Markdown 课程报告 |
| [需求文档.md](./需求文档.md) | 需求分析、角色权限、业务流程和验收标准 |
| [开发流程.md](./开发流程.md) | 开发阶段、协作方式、测试流程和验收清单 |
| [API.md](./API.md) | 后端接口约定、请求示例和演示数据脚本说明 |
| [deploy.md](./deploy.md) | 环境要求、数据库初始化、前后端启动和验证命令 |
| [项目完成情况.md](./项目完成情况.md) | 当前完成度、验证结果、剩余风险和验收建议 |

## 材料目录

| 目录 | 说明 |
| --- | --- |
| [api-tests/](./api-tests/) | curl/API 测试结果 JSON、自动化测试索引和数据库验证记录 |
| [screenshots/](./screenshots/) | 验收截图，当前包含前端主流程、数据库数据、后端测试和前端构建截图 |
| [design/](./design/) | 设计说明、视觉 QA 和原型图素材 |
| [course/](./course/) | 课程提交材料，包含任务书、课程报告、答辩 PPT 和提交说明 |

## 提交前检查

- 确认课程报告封面中的班级、姓名、学号和指导教师已填写。
- 确认 `screenshots/` 中包含登录、注册、提交报修、工单详情、管理员派单、维修处理、学生评价、统计页、数据库数据、后端测试和前端构建截图。
- 确认 `api-tests/INDEX.md` 中记录最终回归结果。
- 确认根目录 `README.md` 的启动命令和演示账号与当前实现一致。
