# 项目文档索引

本目录集中整理课程验收相关材料。完整项目文档仍保存在 `docs/` 目录下。

## 课程提交材料

| 文件 | 用途 |
| --- | --- |
| `答辩.pptx` | 答辩展示 |
| `软件zy2301_吴家乐.docx` | 正式课程报告 |
| `课程报告.md` | 课程报告 Markdown 备份 |
| `README.md` | 本目录提交说明 |
| `答辩演示流程.md` | 现场演示步骤和答辩问题准备 |

## 项目说明类文档

| 文件 | 用途 |
| --- | --- |
| `项目说明-README.md` | 项目总览、功能和运行方式 |
| `需求文档.md` | 需求分析、角色和功能说明 |
| `开发流程.md` | 开发阶段、实现过程和协作记录 |
| `项目完成情况.md` | 功能完成度和验收状态 |

## 技术与测试文档

| 文件 | 用途 |
| --- | --- |
| `API接口文档.md` | REST API 接口说明 |
| `部署运行说明.md` | 环境要求、数据库初始化和启动说明 |
| `接口测试记录.md` | 接口测试索引和测试结论 |
| `screenshots/` | 页面、数据库、测试和构建截图 |
| `txt/` | 主要项目文档的 TXT 格式副本 |

## 源码与数据库位置
