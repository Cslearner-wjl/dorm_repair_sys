## 3. 已完成能力

### 3.1 后端

- Spring Boot 3 + Java 17 项目结构完整。
- MyBatis XML Mapper、实体、DTO、VO、Service、Controller 分层清晰。
- JWT 登录鉴权、`@RequireRole` 角色权限校验和统一响应 `ApiResponse<T>` 已实现。
- 学生报修：新增、我的报修、详情、修改待受理工单、撤销、确认、评价。
- 管理员：全部工单、审核、驳回、派单、用户管理、软删除、统计。
- 维修人员：我的任务、开始处理、提交完成、申请重派。
- 工单详情包含处理时间线和评价信息。
- 图片上传接口 `/api/uploads/images` 和 `/uploads/**` 静态资源映射已实现。

### 3.2 前端

- React 19 + TypeScript + Vite 项目可构建。
- Axios API 封装已覆盖认证、报修、用户、统计和上传接口。
- 登录/注册、学生首页、提交报修、我的报修、进度跟踪、工单详情、管理员首页、派单、用户管理、个人中心、公告页已拆分到 `pages/`。
- 通用布局、弹窗、状态徽章、分页、筛选、图表、时间线等组件已拆分到 `components/`。
- JWT token 本地保存、请求拦截、角色首页跳转和错误提示已实现。
- 图片上传控件已接入上传接口。
- 注册模式已隐藏演示账号切换，并明确公开注册仅面向学生账号。

### 3.3 数据库与演示数据

- `sql/init.sql` 可创建 `dorm_repair` 数据库及四张核心表；`sql/demo-data.sql` 载入完整验收演示数据。

- `sql/demo-data.sql` 覆盖全部 9 种状态（PENDING / APPROVED / ASSIGNED / PROCESSING / WAIT_CONFIRM / COMPLETED / REJECTED / CANCELLED / NEED_REASSIGN）的演示工单。
